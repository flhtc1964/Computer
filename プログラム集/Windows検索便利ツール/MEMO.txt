K-Cc（Windows検索便利ツール）

最新バージョン： Ver.2016-01-21 

開発名【　KUHEN_W_CTRL_C　】


作者： SKojima@kitahama.or.jp


ソフトは、VB .NET VS2012を使用しています。

【重要事項】

このソフトは無保証です。
自己責任でお使い下さい。

機能拡張依頼の対応は予定していません。

リモートデスクップの画面では利用出来ません。


【動作環境】

Windows7, Windows10


【インストール方法】

Zip型インストールファイルを前回と同じ場所に上書き解凍し、このプログラムフォルダにある　setup.exe　を実行しインストールしてください。
前回のプログラムが残っていたら、事項の【アンインストール方法】を実施後再度インストールをしてください。


【アンインストール方法】
コントロールパネルのアプリケーション管理から削除出来ます。


【使い方】

１．パソコンにインストールされた【　KUHEN_W_CTRL_C.exe　】を実行
２．「基本モード選択」コンボボックスで利用する検索モードを選択
　　＃基本モードを選択後、追加で使いたい検索にチェックON可能

「Google」「英辞郎」「weblio英和」「Googleマップ」利用可能


【キーボード】で検索

１．検索したい文字をマウスまたは、Shift＋矢印キーで選択
２．Ctrl + C を２回押す

１．検索したい文字をマウスまたは、Shift＋矢印キーで選択
２．Ctrl + C を１回押す
３．Ctrl + Insert を１回押す
　　＃勝手に上書モードになるので現在の状態を表示するチェックを利用可能

１．検索したい文字をマウスまたは、Shift＋矢印キーで選択
２．Ctrl + C を１回押す
３．Shiftボタンを少し長押し気味で１回押す


【マウス】で検索

１．検索したい文字をマウスまたは、Shift＋矢印キーで選択
２．現画面でマウスの右ボタンをダブルクリック
＃アプリによって動かない（選択文字を取得出来ない）が動けばOK

１．マウスで文字列を選択
２．Ctrl + C を1回押す
３．現画面でマウスの右ボタンをダブルクリック
＃この方法は確実
